# Averypen Quant Research & Systems Portfolio

This portfolio presents a public, sanitized overview of systematic research and quant systems architecture work.

The focus is not to disclose proprietary trading signals or live trading configuration. The focus is to demonstrate research-to-production design across data, signals, portfolio construction, risk controls, execution lifecycle, auditability, and reproducibility.

## Public Project Map

| Repository | Purpose |
|---|---|
| `ares-public-architecture` | Synthetic-data demonstration of a US single-name equity profit engine architecture |
| `vnq-datalake-public-manual` | Public manual for a reproducible equity research data lake |
| `chronos-master-portfolio-demo` | Planned master portfolio governor demo |
| `vhedge-risk-allocation-demo` | Planned defensive multi-asset risk allocation demo |
| `equity-alpha-research-lab` | Planned US equity factor research framework |
| `qinglong-ashare-research-lab` | Planned A-share market-structure-aware research framework |
| `quant-execution-audit-toolkit` | Planned dry-run execution and audit lifecycle toolkit |

## Core Principles

- Synthetic data by default
- Dry-run execution by default
- No proprietary signals
- No live broker configuration
- No account identifiers
- No investment advice
- Reproducible runs with config snapshots and manifests
- Explicit risk controls before return claims
